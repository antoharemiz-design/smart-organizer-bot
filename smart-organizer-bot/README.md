## ✨ Возможности

| Функция | Описание |
|---------|----------|
| 🧠 **AI-понимание** | Понимает «напомни», «запланируй», «к 7 вечера» |
| 🎤 **Голосовой ввод** | Транскрибация через Whisper (локально, бесплатно) |
| 📝 **Задачи** | Создание с датой, временем, периодичностью |
| 📅 **Календарь** | Inline-календарь для просмотра задач |
| 🔔 **Напоминания** | Уведомления точно в указанное время |
| 🌅 **Утро** | Ежедневный план на день в 08:00 |
| 🌙 **Вечер** | Опрос о выполнении с кнопками в 21:00 |
| 📊 **Статистика** | Продуктивность, сброс |
| 📤 **Экспорт** | Выгрузка в .ics (Google/Apple Calendar) |
| ⚙️ **Настройки** | Своё время утра/вечера |
| 🔒 **Без дубликатов** | Не создаёт повторяющиеся задачи |
